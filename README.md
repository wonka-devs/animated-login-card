# Animated Glassmorphism Login Card

A responsive login UI with glassmorphism design and animated background elements, built with pure HTML and CSS.

## Features
- Glassmorphism card effect
- Animated floating background circles
- Responsive layout (mobile-friendly)
- Smooth hover and focus transitions
- Custom checkbox styling

## Tech Stack
- HTML5
- CSS3 (animations, backdrop-filter, flexbox)
